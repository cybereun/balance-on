# 밸런스 온 구현 계획

**목표:** 모바일 카메라·관절 스켈레톤 기반의 설치형 균형 운동 MVP를 Vercel에 배포 가능하게 만든다.

## 완료 항목

- [x] React + Vite 정적 앱과 Vercel 설정
- [x] 설치용 매니페스트, 브라우저/설치 아이콘 구성
- [x] MediaPipe Pose Landmarker 카메라 루프
- [x] 3종 운동의 간단한 상태 판정과 실시간 코치 문구
- [x] 운동 기록의 브라우저 로컬 저장
- [x] 프로덕션 빌드 점검

## 수동 확인 항목

- [ ] 실제 휴대폰 HTTPS 주소에서 카메라 권한 허용
- [ ] 전신이 프레임에 있는 상태로 3개 운동 판정 확인
- [ ] 브라우저 메뉴에서 홈 화면 설치 확인
